== Description ==
**Human Tagger** is a lightweight WordPress plugin that allows you to create and manage a `humans.txt` file directly from your WordPress dashboard. Seamlessly integrates with the Site Editor (FSE) to add a `<link rel="author">` tag in your site's `<head>` section, linking to a customizable `humans.txt` file.

== Features ==
- Easily create and edit a `humans.txt` file from the WordPress admin panel.
- Automatically adds `<link rel="author" href="/humans.txt">` to your site's `<head>`.
- Full compatibility with the Site Editor (FSE).
- Lightweight and optimized for performance.

== Installation ==
1. Upload the `human-tagger` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Settings > Human Tagger** to customize your `humans.txt` file.

== Frequently Asked Questions ==

= What is a `humans.txt` file? =
The `humans.txt` file provides information about the people behind the website, including developers, designers, and other contributors.

= Does this plugin affect my site's SEO? =
While it doesn't have a direct SEO impact, it can enhance trust and transparency, indirectly contributing to better user engagement.
